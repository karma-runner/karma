Designed for quick development.

Relying on the best ones - Node.js & Socket.io.

Open Source

Developed for and maintained by open source community at GitHub. Feel free to fork the repository and send us pull request with some cool improvements.

Remote Control

Control the whole workflow from command line or your IDE.

Test on Real Devices

Test your code on real browsers and real devices such as iPhone, iPad, Android.

Easy Debugging

Easy debugging directly from your IDE (WebStorm).

Jasmine Integration

Testacular is framework agnostic - you can run any JS code. But we provide an adapter for Jasmine. Adapters for more testing frameworks coming soon.
